# gate_bot

Minecraft-бот на основе [mineflayer](https://github.com/PrismarineJS/mineflayer) для поддержания активности Aternos-сервера и взаимодействия через чат.

## Особенности

- Автоматическое нажатие на калитку для предотвращения отключения Aternos-сервера
- Поддержка навигации по координатам через Baritone
- Простые команды в чате на русском языке
- Работа в офлайн-режиме

## Установка

```bash
npm install mineflayer @miner-org/mineflayer-baritone vec3
```

## Запуск

```bash
node main.js <host> [port] [username]
```

Примеры:
```bash
node main.js example.com          # порт 25565 по умолчанию
node main.js example.com:12345    # с указанием порта
node main.js example.com 25565 MyBot  # с именем бота
```

## Команды в чате

| Команда | Описание |
|---------|----------|
| `@бот привет` | Приветствие и информация о боте |
| `@бот калитка` | Запустить авто-клик по калитке перед ботом |
| `@бот стоп` | Остановить работу с калиткой |
| `@бот иди x y z` | Переместиться к указанным координатам |
| `@бот помощь` | Показать список команд |
| `@бот уходи` | Отключить бота от сервера |

> Замените `@бот` на имя вашего бота. Упоминайте его в сообщении, чтобы он отреагировал.

## Требования

- Node.js 16+
- Minecraft-сервер в офлайн-режиме (или настройте `auth` в коде)
